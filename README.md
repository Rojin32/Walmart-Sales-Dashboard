# Walmart Sales Performance Dashboard

## 📊 Project Overview
Analyzed Walmart sales data to identify store performance, seasonal trends, and holiday impact.

## 🔧 Tools Used
- Python (Pandas, Matplotlib)
- Power BI

## 📈 Key Insights
- Sales peak during mid-year months
- Top stores contribute majority of revenue
- Non-holiday periods generate consistent sales
